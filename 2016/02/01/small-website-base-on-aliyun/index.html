基于阿里云的小网站结构

2月 01, 2016 发布在 云计算

  • 「阿里云ECS」的磁盘不支持共享，可以用「对象存储OSS」放一些用户上传的东西，通过每个「阿里云ECS」进行读取写入。
  • Session共享
    • Spring Session 支持 Redis，与「云数据库Redis版」搭配使用
    • memcached session manager 只可以在Tomcat上使用，与「云数据库Memcache版」搭配使用
  • 「云数据库RDS」支持读写分离，暂时不太了解
